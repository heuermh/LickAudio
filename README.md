# Lick Audio
Virtual Eurorack Modules for [VCV Rack](https://vcvrack.com)

## Modules ##

### Atan ###

Wave shaper, atan(x)/(&pi;/2)

Distortion, Waveshaper

### Fabs ###

Wave shaper, x/(1.0 - fabs(x/amount))

Distortion, Waveshaper

### Tanh ###

Wave shaper, tanh(x)

Distortion, Waveshaper

### Tick ###

Static clock generator

Clock, Clock generator
